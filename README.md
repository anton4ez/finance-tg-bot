# Finance Analyst Bot

Бот запущен и доступен по ссылке: https://t.me/finance_managerr_bot

Данный проект развернут на удаленном VPS сервере и работает 24/7.

## Описание проекта
Telegram-бот для автоматизированного учета личных финансов, анализа текущих расходов и генерации аналитических отчетов. Система использует NLP-алгоритмы для интеллектуальной классификации трат, построена на асинхронной архитектуре и использует PostgreSQL для надежного хранения данных.

## Функциональные возможности
* **Умная категоризация:** Автоматическое определение категории расхода на основе текстового описания (NLP).
* **Учет транзакций:** Удобное добавление записей через систему состояний (FSM).
* **Визуализация:** Автоматическая генерация круговых диаграмм (Matplotlib) в headless-режиме для корректной работы на сервере.
* **Экспорт данных:** Выгрузка полной истории транзакций в формате Excel (.xlsx) через Pandas.
* **База данных:** Асинхронное взаимодействие с PostgreSQL через SQLAlchemy 2.0.

## Технологический стек
* **Язык:** Python 3.12+
* **Фреймворк:** Aiogram 3
* **ORM:** SQLAlchemy 2.0 (Async)
* **БД:** PostgreSQL
* **Аналитика и ML:** Pandas, Matplotlib, PyTorch (CPU-version), Sentence-Transformers
* **Деплой:** Docker, Docker Compose, Ubuntu Server, systemd

## Запуск через Docker (Рекомендуемый способ)

Проект полностью контейнеризирован. Для быстрого запуска вам понадобятся только Docker и Docker Compose.

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/anton4ez/finance-tg-bot.git
   cd finance-tg-bot
   ```

2. Создайте файл .env в корне проекта. 
   ```env
   BOT_TOKEN=ваш_токен
   DB_HOST=db
   DB_PORT=5432
   DB_USER=postgres
   DB_PASSWORD=ваш_пароль
   DB_NAME=finance_bot
   ```

3. Соберите и запустите контейнеры в фоновом режиме:
   ```bash
   docker-compose up -d --build
   ```

Система автоматически скачает образ PostgreSQL, соберет образ бота с нужной версией Python, установит зависимости и свяжет их в единую виртуальную сеть.

## Ручная установка (Без Docker)

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/anton4ez/finance-tg-bot.git
   cd finance-tg-bot
   ```

2. Настройте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Linux/macOS
   venv\Scripts\activate     # Для Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Создайте файл .env в корне проекта и заполните его данными:
   ```env
   BOT_TOKEN=ваш_токен_от_botfather
   DB_HOST=localhost
   DB_PORT=5432
   DB_USER=postgres
   DB_PASSWORD=ваш_пароль_от_базы
   DB_NAME=finance_bot
   ```

5. Инициализируйте таблицы базы данных и запустите проект:
   ```bash
   python reset_db.py
   python main.py
   ```

## Структура проекта
* bot/ — Обработка команд, фильтры и клавиатуры.
* database/ — Модели данных и логика взаимодействия с PostgreSQL.
* analytics/ — Модули формирования графиков и Excel-отчетов.
* main.py — Точка входа в приложение.
* .env — Конфигурационный файл (исключен из репозитория).

## Развертывание
Проект развернут на VPS под управлением systemd и Docker Compose. Это обеспечивает автоматический перезапуск контейнеров при сбоях и гарантирует доступность сервиса в режиме 24/7.
